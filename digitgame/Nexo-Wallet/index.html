Nexo Wallet

Abstract ▹ Nexo抵押借贷方式提现数字资产


Nexo Wallet, 创新理念的 “法币<->数字资产” 的方案, 通过抵押借贷的方式，实现两者的兑换。Nexo公司主要的收益，是借贷的利息 (按天，随时偿还)。
数字资产扣除已用的额度，剩余的可随时提现至其他数字钱包，0手续费 (含网络费)。
存储的特定币种或stablecoin有存款利息。

 

1. Nexo Token

 

  • 基于ERC20
  • Nexo Token, 持有分红30%
  • Changelly等小众交易所，可以兑换

  • Nexo Token优惠利率：如持有Nexo Token, 享有5.9%年利率(即万分之1.6/天)的借贷利率 (其他代币: APR@11.9%); 偿还利率时，可使用Nexo Token抵扣。 『使用Nexo Token来偿还利息，即可享用5.9%年利率。』

      当使用NEXO作为抵押或还款时，利息折扣适用。当使用NEXO代币偿还贷款或由NEXO代币支持的部分贷款（两者不能合并）时，
  年百分比率（APR）从5.9％开始。不使用NEXO代币时的年百分比率为11.9％。

 

2. 特定法币和StaleCoins资产 存入收益@8%年利率

 

  • 8%年利率
  • USD/EUR/GBP/TUSD/USDT/USDC/PAX/DAI

 

3. StaleCoins资产

 

  • 借贷消费或提现至银行时(即当作法币提现、消费、偿还借贷时)，承诺与美金1：1价值。
  • 当作数字资产提现时(如提现至crypto wallet)，按照当下的市价。

 

4. 数字资产最低提现数量

 

  • 提现手续费(含网络费)：0

 

5. Credit Line提现至银行

 

  • 亲测：
    • Nexo转入TransferWise账户：US ABA Local Transfer (shown from “Currency Cloud”)，T+0 到达账户 (比local ACH要快)。
    • TransferWise转账至国内银联卡：~3小时，手续费：1.5%。
  • 最低提现金额：$500。
  • 借出的贷款，默认期限为1年。
    • 可尝试要求延期；
    • 如果借贷偿还<30天，则仍计30全天的利息；
    • 偿还时，有$2.04还款手续费；
    • 可以提前偿还，没有额外费用；
    • 计息时间，按CET 时间。CET = 冬季时间为 UTC+1，夏令时 UTC+2。
  • 借出的那一刻，即起算该全天的利息interest = Loan Amount * APR/365。(APR: 5.9%@Nexo Token, 11.9%@Others)
  • Transfer Type for Withdraw:
    • Withdraw USD to US bank account: 使用”ABA Local”
    • Withdraw USD to non-US bank account: 使用”ABA Swift”
性价比最高的方案 (仅支付30天的利息(<30天，仍计30全天的利息)): 
1) Credit Line (X) = 充入的数字资产等值价值(Y)的15%~50%，即充入 X/15%~50%的数字资产以获得价值X的Credit Line。
2) 将Credit Line (X)提现至银行账户，从提现那一刻开始起息当天的利息 (Z) = X * APR@11.9%/365。(该利息，逐天加至借出
   本金, 即Z+X)
3) 发起提现成功后，当天即可使用已存入的数字资产Y，进行全额偿还Credit Line (X)。(但利息，(K) = 30天内仍需支付30全天
   的利息)
   『<30天总费用：(n天数*Z+X) + K + $2.04还款手续费』，『>=30天总费用：(n天数*Z+X) + $2.04还款手续费)』
4) "全额偿还"后, 即可将剩余的数字资产提现至其他数字钱包。(NexoApp提现至数字钱包, 没有手续费+没有网络费)
