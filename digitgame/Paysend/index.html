Paysend

Nov 7, 2019

Category▸  DigitGame
Tag▸  LegacyBank   E-Channel

Abstract ▹ 支持Visa Direct和Mastercard MoneySend, 卡到卡汇款, 发往+70个国家


特点：手机号 = 系统判定的”居住国”，ID可以是任何国家，但POA必须=居住国地址。
可以 No KYC，直接汇/收款@最低额度。


Paysend公司性质

  • Paysend为Paysend plc的交易名称，注册地址为英国苏格兰。
  • 企业编号：376020。
  • 注册地址：Cluny Court John Smith Business Park, Kirkcaldy, Fife, KY2 6QJ.
  • Paysend plc受英国金融行为监管局认可并受其监督，编号：900004。

 

费用

  • 每笔(和汇款金额无关)：1英镑/1.5欧元/2美元。
  • Coupons: 截止到Dec 31,2022, 优惠码 SENDFREE, 免除第一笔汇款手续费。

 

认证

  • 填写美国手机号注册登录、中国ID，可以通过认证。
  • 地址：在最初注册时，地址可填写中国，但如需POA,则必须与手机号一致的国家，否则无法通过POA。
  • 款项额度(Amount Limit)基于KYC认证的程度：
    • no KYC (6个月内累计1000美金)
    • ID Verification (6个月内累计5000美金)
    • ID + POA (6个月内累计17000美金)
    • ID + POA + Financial Prove (6个月内累计>17000美金)

 

70个国家

  • 阿尔巴尼亚、安道尔、奥地利、比利时、波黑、保加利亚、克罗地亚、塞浦路斯、捷克共和国、丹麦、爱沙尼亚、芬兰、法国、德国、希腊、荷兰、匈牙利、冰岛、爱尔兰、意大利、哈萨克斯坦、拉脱维亚、立陶宛、卢森堡、马其顿、马尔他，墨西哥、摩纳哥、蒙特内哥罗、圣马力诺共和国、塞尔维亚、斯洛伐克、斯洛文尼亚、西班牙、瑞典、瑞士、乌克兰和英国。
    • 其中，新加坡收款卡暂不支持visa。
  • 不含香港。

 

手机号注册&登录

  • 使用居住地国家的手机号，进行注册和登录。
  • 中国手机号
    • 支持请求汇款：
      • 可以向世界各地超过70多个国家/地区的人发起收款请求。
      • 不可自己请求自己。
    • 不支持汇款服务。
  • 美国手机号
    • 不支持请求汇款。
    • 支持汇款服务：
      • 可以向世界各地超过70多个国家/地区的人发起收款请求。
      • 汇款时，仅接受美国居住地发行的银行卡。

 

汇款&收款方式

  • 汇款
    • 仅支持银行卡方式。
    • 仅限于汇款人所在国家发行的银行卡。
    • 可发款国：from 欧洲、美国。
  • 收款
    • 支持银行卡方式、银行账户方式。
    • 大多数国家Visa/MasterCard；少部分国家仅Visa或仅MasterCard；中国仅银联卡。
    • 收款国：70个国家(没有香港)。
      • 汇款至新加坡Wirecard Visa：显示Visa维萨卡暂不支持，请选择汇款至其他银行卡或常规银行账户。

 

请求汇款

  • 美国不支持请求汇款。
  • 中国支持请求汇款：
    • 自己不可以发给自己。
    • 家人银行卡应该可以发给自己。(待验证)
    • 可以向世界各地超过70多个国家/地区的人发起收款请求。

 

支持的银行卡类型

  • 欧洲、英国、美国、土耳其和以色列 — Visa/Mastercard
  • 俄罗斯和哈萨克斯坦—维萨卡 — Visa/Mastercard
  • 中国-银联卡 (UnionPay)

 

Q&A

 

总结

  • 中国手机号注册的账号：不可汇款；请求汇款也不可自己给自己 (除非家人给自己)。
  • 美国手机号注册的账号：支持汇款，汇款时仅接受美国居住地发行的银行卡；不支持请求汇款。
