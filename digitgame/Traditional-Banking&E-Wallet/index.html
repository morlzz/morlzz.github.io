    TAP N GO拍住赏, AlipayHK。

 

  • 香港发行的预付卡

    1) EPayLinks 易票联

    -“易票联”所发行的卡包括：全球付、爱汇旅支卡、安游、neat；
    -所有卡均可在”全球付 Web Portal“和”全球付微信公众号“登录; 但”全球付App”，仅”全球付”卡号可登录。
    -香港 FPS (无费用)：Epaylinks 银行代码 930 + 卡号 + 姓名 (各渠道购买认证时的姓名)。
    -“条款 & 细则 “中，对于 “隐私 /用户数据”比较 Unfriendly (默认为在其所有子公司内部共享 + 营销 + 应法律部门要求 (除非法律所禁止); 如销户，未提及 “永久删除用户数据”)。

     

    • EPayLinks 易票联 – 全球付  (当”实名认证”要求补充额外的”隐私数据”时，建议销卡)

      全球付，支持绑定内地银联卡, 直接充值，实名认证后可进而使用 香港 FPS 转账至任何的港币账户。

      • 缺点：实名认证比较麻烦，稍不慎会要求补充一堆敏感资料。(考虑其很差的”隐私保护策略”，可果断销卡)
      • 好处：
        • 除了首充300CNY/350HKD/50USD，再次充值没有金额限制。
        • 转帐：可直接转帐至”国内银联卡”@1%手续费。(转账有时间限制：7am~8pm) (比”爱汇的兑回”性价比高很多)。 『实测 - 全球付人民币账户：仅人民币账户-to-人民币账户』
        • 充值：”银联卡”直充@1%手续费。 『实测 - 全球付人民币账户：充值可港币/可人民币』
      • 外汇管制：不受”外汇管制”限制, 但有”限额表规定“。

     

    • EPayLinks 易票联 – 爱汇国际旅支卡 (Details)   (已销卡)

2. Singapore E-Wallet/Prepaid Card

  • 新加坡发行的预付卡

    1) 携程环球现金卡

    携程环球现金卡，支持7种币种：美元、日元、港币、欧元、新加坡元、澳元、英镑；
    CNY直充 + 无充值手续费 + 7种币种间自由免费转换；
    但仅支持”消费/取现/网购”，无法转账，最低500CNY等值外币充值金额；
    账户管理费：6个月未使用或者未激活，则第7个月开始将收取1.5美元/月的管理费用。

 

3. 中国大陆

  • 中国大陆扫码支付、闪付

    概况汇总：线下支付(扫码支付、闪付)、线上支付(银联在线、快捷支付)。

 

  • 中国大陆手机银行/网银 - 柜面注册 vs 自助注册, 权限不同

    • 柜面注册 vs 自助注册：
      • 在手机银行/网银上，自己申请添加银行卡，称为自助注册用户；
      • 自助注册用户 只能操作查询余额、注销、冻结银行卡等，无法进行资金转账支出等操作。
      • 柜面注册用户 需本人携带银行卡、身份证，到营业网点柜面开通，开通后就可以在手机银行上进行转账操作。

     

    • “银行卡转账功能”的折中方法：
      • 如果不想去柜面开通，但有转账的需求，可以使用“银联云闪付”App中的转账功能、以及微信/支付宝的快捷支付功能；
      • 该”折中方法”, 实则使用的是”线上支付(银联在线、快捷支付)“功能，和”各家银行自家的网银/手机银行”是两条不同的路径。

 

  • 中国银行 - 长城跨境通国际借记卡

    目前国内银行渠道唯一发行的国际借记卡，有Visa/MasterCard两种选择。

    • 支持19种外币：美元、欧元、日元、港币、英镑、加拿大元、澳大利亚元、瑞士法朗、新加坡元、澳门元、韩国元、丹麦克朗、挪威克朗、新西兰元、菲律宾比索、俄罗斯卢布、瑞典克朗、泰国铢、南非兰特共计19种外币；不支持CNY。
    • 年费：金卡 10 CNY/年；白金卡 20 CNY/年。
    • 无账户管理费。
    • 境外ATM取现：
      • 手续费标准为取现金额的0.5%，最高5美元，最低3美元。
      • 2019年支持每月首笔境外ATM取现免收手续费。
    • 据说：
      • 如果消费此卡会先扣105%，5%待入账以后会返还（防止汇率波动）；
      • 此卡境外取现，受人行管控，每日不超过等值1万元人民币，一年不超过10万同名共享。

 

  • 内地银行间 - 外币划转

    • 本人不同银行间”外币划转”
      • 只能柜台办理。
      • 划转通道：周一~周五 9:00~16:00; 如周末办理，资金会pending，直至周一”划转通道”工作时间。

     

    • 来自他人的”外币划转”
      • 只能直系亲属，需关系证明。

     

    • “外币划转”手续费
      • 中行：CNY收费，最低50元或者千分之一 * Amount，最高260元。

 

  • 内地手机银行(安卓版)APK官网直接下载链接

 

4.全球支付平台

  • PayPal

    PayPal US, PayPal HK, PayPal China；风控；绑卡/账户。

 

  • Western Union

     

    • Western Union China

       

      • 在线汇款 - 只能透过合作银行在线发汇，如邮储银行；
      • 在线收款 - 透过合作银行在线收款 (如邮储银行)、或者直接汇入中国境内合作银行的银联卡。

     

    • Western Union HK

       

      • 在线汇款 - Western Union HKApp, 支持在线直接发送款项。
        • 支持”信用卡/借记卡/银行转账”的方式支付汇款；
        • 支持”电子钱包”的方式支付汇款：和Tap N GO拍住赏，深度合作，WU HK手机App直接调用拍住赏 (金额：100~2000 HKD)，无手续费，汇率及其优惠。

     

    • Western Union SG

       

      • 无需身份认证；
        • 未认证，最高发送金额：1000 SGD/每次；
        • 认证，最高发送金额：5000 SGD/每次；
      • 在线汇款 - Western Union SGApp, 支持在线直接发送款项。
        • 支持”信用卡/借记卡/银行转账”的方式支付汇款；无手续费。

 

  • TransferWise

    TransferWise, 开通外币银行账户(borderless account), 可直接提现至国内银联卡号 (自动转成 RMB@中间汇率, 31,000 CNY/每笔)。

    • 相较Payoneer, TransferWise 对于资金来源较为宽松，广泛支持个人金额的中转、以及数字资产等来源，都支持自动转成RMB，直接银行转账至银联卡号 (即国内的 TransferWise 合作方直接将 RMB 转账至客户的银联卡号)。
      • TransferWise CNY转账至银联卡，实际上走的是与银联合作的 银联跨境汇款服务: 银联全球速汇(MoneyExpress)。
    • TransferWise 不支持 EPay。
    • TransferWise 不支持 ACH Dedit
      • 例如：绑定PayPal进行验证时，PayPal可以ACH Deposit但无法ACH Dedit，因而无法绑定PayPal。

 

  • ADVCash

     

    ADVCash，英国公司; 法币<->数字货币; 支持虚拟卡/实体卡; 支持China; KYC + POA。

    • 特点：
      • 法币<->数字货币 非常灵活；
      • 支持数字资产、Credit Card 充值/提现：
        • Credit Card 充值 (fee@2.99%)/ 提现(fee@6 USD/EUR + 1.99%)；
        • 数字资产，使用他们内部的转化率。
      • 整体ADVCash系统运作/运营的体验。(推荐使用！)
    • 币安合作钱包：目前直接支持充入的法币币种 – 欧元，英镑，俄罗斯币等。(人民币属于场外交易(自动匹配买/卖家))

 

  • Paysend

     

    -Paysend 英国公司; 支持Visa Direct和Mastercard MoneySend, 卡到卡直接汇款；
    -可发往+70个国家; 目前，中国等仅可”请求汇款”，美国等仅可”汇款”; 暂向不支持”香港”、”新加坡”Visa卡汇款;
    -Amount Limit varies on KYC level (no KYC/ ID Verification/ ID + POA/ ID + POA + Financial Prove);
    -最优的汇款手续费：每笔(和汇款金额无关)：1英镑/1.5欧元/2美元。
