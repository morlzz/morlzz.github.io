Arba Cryptocurrency Wallet

Abstract ▹ Arba数字货币钱包


Like & recommend this “Arba bitcoin wallet”. 以下整理为原创。

优点：

  • 初心：vision - 实现100%去中心化的数字银行。👍

  • 简单：前台的设计简单易用，看不到”技术”的影子。

  • 便利：

    • 30种数字货币 + 52种法币。
      • 支持 native 四种数字货币：btc/bch/lte/eth (即支持这四种的原生钱包地址) 。
      • 其他26种数字货币和52种法币: 通过app内的exchange“合成”对应币种。
      • 所有exchange，实则采用“smart contract”，即锁定等值的原native数字币种。
        • 官网案例：btc通过”euro contract”转换成euro，之后用户想将euro转换成eth，实际过程是 “convert contract”：”redeem euro contract” first -> “fix value of btc” to the amount of eth = “create eth contract”.
        • smart contract：保存在区块链，因此arba需要支付network mining fee，arba将此费用折摊到exchange rate转换率中。
          • 即，”arba spread on exchange rate”包含两部分：1) 实际发生的 network fee; 2) abra宣称唯一用来盈利的转换率差价(由arba自己设定)。


    • 所有币种市场行情，一目了然，行情价格可以参考(接近全网较低价格水平)。

 

  • “数字银行”, 打通数字货币和法币的进程：
    • 目前，美国和菲律宾，可以自由在”数字货币、法币”之间, deposit、withdraw from/to 传统银行。
    • 其他国家，操作上暂无法和传统银行挂钩。
    • 其他国家 (部分)，可通过abra合作伙伴simplex，使用visa card 直接购买btc, simplex收取5%手续费(min. 10usd)。
      • simplex 优势：限额空间大；没有身份认证等要求。

 

  • 安全：
    • 私钥/recovery phrase 完全由用户掌握(non-custodial wallet)，arba 无法得知。
    • HD wallet，每次deposit数字货币后，钱包地址变自动变更，好处：保护隐私，(旧的依旧生效)。

 

  • 良心：
    • zero fee: mining fee for withdraw、no fee for deposit、zero fee for transfer inbetween arba a/c.
    • only fee from “spread on exchange rate”: Arba在币种间转换(exchange)时, 从转换率上获利。
