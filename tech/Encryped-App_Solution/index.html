加解密软件/方案

Feb 19, 2019

Category▸  Tech
Tag▸  Secure

1. Cryptomator

  • APP: pc版free，mobile版付费

  • 方案：cryptomator加密 + 储存 (云端 OR 本地)


 

  • 加密方式：Passphrase为用户自己设定的256位密钥，只有自己知道

 

 Note: scrypt，比sha256哈希加密，更加耗硬件资源的哈希算法。可参考链接。


2. KeePass

  • APP: pc版官网

  • APP: mobile版, Keepass2Andriod, by Philipp Crocoll(Croco Apps)

  • Browser Plugin: Firefox, Kee, by Luckyrat

  • Browser Plugin: Chrome, KeePassHelper Password Manager, by unixeco
