华为荣耀7 (PLK-AL10全网通)解BL锁/刷TWRP

1. 手机状态

几年前曾经解锁过BootLoader，显示”fastboot unlock”, 但是当通过“fastboot命令”刷入TWRP时，显示remote: Command not not allowed 错误, 无法刷入TWRP。对此，网络的建议：重新解锁BL一次。

 

2. 解BL锁

刷入官方以外的任何东东 (如TWRP/第三方固件)，前提条件：必须解BL锁。

  • 2018年，华为官方已正式停止申请BL解锁码。

    2018年后，需要BL解锁码的方式有两：

    • 淘宝等收费网站
    • 网上流传的方法，自行获得解锁码
      亲测体验：
      • 必须root先，EMUI 4.x.x 很难root，降级至B220或B201 (EMUI 3)后，使用kingroot.apkroot成功。
      • 使用以上链接方法尝试：搜索WVLOCK，得到的是UUUUU......UUUDDDD.......DD，并非16位解锁码，无解。
        也许和”fastboot unlock”状态有关，也许是针对”曾经解过锁但需重新解锁”所特有的错误，maybe, not sure.


      以上链接涉及的命令：

        adb shell
  $ su
  # cd /storage/sdcard0/
  # mkdir -p PartitionBackup
  # cd /storage/sdcard0/PartitionBackup

  # dd if=/dev/block/platform/hi_mci.0/by-name/nvme of=/storage/sdcard0/PartitionBackup/nvme
  # dd if=/dev/block/platform/hi_mci.0/by-name/oeminfo of=/storage/sdcard0/PartitionBackup/oeminfo
  # dd if=/dev/block/platform/hi_mci.0/by-name/recovery of=/storage/sdcard0/PartitionBackup/recovery

  root@hwp7:/storage/sdcard0/PartitionBackup # strings nvme | grep WVDEVID -B 1

 

3. 其他尝试：成功刷入root + TWRP

无心插柳柳成荫，在B388官方固件状态下，尝试使用YoRoot_V1.1进行root。成功root的同时，YoRoot还自动成功刷入TWRP 2.8.7.0. (之后可手动更新TWRP版本)

Amazing! (至于为什么没有报错：remote: Command not not allowed，未解之迷 :) )，anyway，”fastboot unlock” was still helpful to achieve root/TWRP flashing.

Note: YoRoot_Test_2.0 不支持荣耀7。

 

4. 刷入第三方固件

目标：刷入”论坛优化版固件”，基于极客版(即开发版)进行优化的论坛第三方固件，比较基于原官方固件

  1. 进入TWRP，清除缓存、清楚data.

  2. 刷入论坛优化版固件(zip文件) (一般均含有root和xposed).
    荣耀7 Final极客优化版: PLK-AL10_7.5.16_90208Em

  3. (可选) 待进入系统平滑稳定后，可进入TWRP，刷入更新版本的TWRP.img和supersu.zip (为避免版本不对，xposed.zip for HW 建议不更新).

 

5. 关于荣耀7 官方固件降级/升级说明

  1. 荣耀7 官方final稳定版为B389，YoRoot_V1.1不支持该版本，因此需退回到B388.

  2. 不能直退，需安装华为官方助手 hisuite，点”系统升级”，可降级至B220.

  3. 降后，再通过”fastboot_all线刷方式”或者“dload”卡强刷方式，升级至B388.
