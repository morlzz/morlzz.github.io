Install 4 OS in Single Laptop (win/ubuntu/CloverMacOS/RemixOS)

May 15, 2018

Category▸  Tech   个人经验分享
Tag▸  OS

Abstract ▹ 电脑上同时安装多个OS,步骤/工具/分区要求

本着知识取之于网络，再回馈于网络的初衷，分享个人经验以及将一些信息梳理总结，所有资源来自于网络，再次感谢所有的网络分享者。


电脑：laptop，神舟战神K350S-i7 D1

硬盘现状：2个，HDD (disk0)+SSD (disk1)，UEFI+GPT

Target OS：windows/ubuntu/Clover MacOS/Remix OS


Now，Start:


1. Windows 10:

-由于windows是主用OS，因此将windows放在SSD中。

-由于GPT分区表，默认原版windows安装，除了系统分区，自动分配MSR、ESP、Recovery三个分区，利用dos命令：diskpart，可将ESP

size调大（默认仅100M），这里取600M （具体命令参数网络搜索）。

-如果仅自动分配MSR一个分区，没有关系，先行安装windows，安装完成后，进win系统，继续采用

diskpart命令，手动分配ESP、Recovery分区。

-windows安装U盘制作：采用“rufus.exe”程序，刻录官方镜像到U盘即可。


2. Ubuntu (Linux):

ubuntu安装U盘制作：同样采用“rufus.exe”程序，刻录官方镜像到U盘即可。

-系统文件分区： ubuntu /, 放在SSD，20G，必须设为“主分区 Primary”

-交换分区：swap, 放在HDD，~8G (up to you), 逻辑分区 Logical

-home分区： ubuntu /home, 放在HDD，~80G (up to you), 逻辑分区 Logical

-启动分区：ESP，因为安装win时，已经建立ESP，因此无需再建，共用即可。

-在下面“启动设备的分区”处：选择之前已在disk1建立的ESP，如这里”sdb3“。即可成功安装，ubuntu启动菜单会自动增加windows启动项。

-如需修改启动菜单内容，以下3个文件是关键：

------------------------------------------------------------------------------------

/boot/grub/grub.cfg    它在每次update-grub后自动生成。

/etc/default/grub        改变引导菜单外观的主要配置文件

/etc/grub.d/               各种用于生成grub.cfg的脚本文件，每次update-grub时会执行里面的文件

------------------------------------------------------------------------------------

3. MacOS (Unix): 

Clover MacOS, 仅供个人学习参考，提倡支持原版白苹果。


安装步骤，参考：

http://www.360doc.com/content/18/0915/17/41461333_786915598.shtml


最新含Clover镜像文件:

https://imac.hk/macos-mojave-10-14-18a391-clover-install.html


Note: 

a) MacOS计划装在Disk0 HDD上某个分区上（分配100G空间），要求该分区对应的HDD上必须要有ESP分区，才可以成功“抹除”并格式化成APFS分区，继而才能安装Mac系统。

b) 因现有ESP是在DISK1的SDD上(sdb3)，所以无法给HDD上的分区共用，因此（在安装MacOS前）需提前在HDD上建立一个200M ESP (sda4)（最小200M）。

c) Clover EFI安装后（集成在Clover Configurator），表示电脑硬盘有了启动MacOS的能力。继而，将EFI/Clover目录 全部拷至sdb3/EFI目录下。最后，使用EasyUEFI，新增EFI/Clover/Clouver64.efi为启动引导项。



     EFI目录结构


4. Remix OS (Andriod x86)

HDD上分配一个分区，然后采用自带的安装器和镜像直接安装即可。

- 如需修改启动菜单设置：disk1 ESP中：/boot/grub/grub.cfg文件

++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

UEFI启动菜单配置：采用"EasyUEFI"程序，添加/删除/调整启动菜单。

++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

OS安装完成，共用disk1 ESP分区(sdb3)启动：

描述：Remix OS

分区编号：3

文件路径：/EFI/RemixOS/bootx64.efi

------------------------------------------------------------------

描述：Windows 10

分区编号：3

文件路径：/EFI/Microsoft/Boot/bootmgfw.efi

------------------------------------------------------------------

描述：ubuntu_555

分区编号：3

文件路径：/EFI/ubuntu/shimx64.efi

------------------------------------------------------------------


All done, nice day!
