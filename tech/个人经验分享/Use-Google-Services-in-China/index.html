要点：如何使用Google服务

Sep 29, 2018

Category▸  Tech   个人经验分享
Tag▸  Cloud/VPN/VPS   Google

Abstract ▹ 添加中国信用卡,谷歌云GCE



自从谷歌离开中国后，国内使用Google服务非常的困难，原因有二： 1）国内高墙； 2）谷歌地域隔离“中国”（no china option）。


Step1: 临时梯子（自行找一个临时可用的）


Step2: 关于在谷歌billing profile, 添加不了国内信用卡

  • 注册一个谷歌账户 (需梯子)，但真正深入使用谷歌的服务，还需建立一个“billing profile”，问题来了：

    • 建立billing profile，则需要添加国际信用卡，而国际信用卡账单地址必须写真实的地址，因为谷歌严谨的反欺诈验证机制，写错是无法通过的，IF“写对”–“中国”又不在服务范围之内，故根本没有“中国”选项。(chick & egg problem)

    • 解决方法：写邮件和谷歌客服沟通，参考该方案。

  • 沟通后，再次添加账单地址时，会发现增加了“中国”选项，即表示可以成功添加自己的信用卡了。

  • Note

    • 注册谷歌账户时，“location 位置”建议选择“美国”（梯子也最好连美国），因为如果选择“中国”部分服务功能会受限。

    • “shipping address 运送地址”，可以填写美国的属于你的转运地址（部分商品只能通过转运）。


Step3: 谷歌云GCE

首次启用谷歌云GCE，第一年免费赠送300美金credit，用于搭建VPS。
需要对应“billing profile”，一年后300美金失效并暂停对应云服务，但不会主动从信用卡扣除费用，除非我们主动确认并选择“继续使用”。


搭建方法: 网络自行搜索。

简略设置

  1. 登陆谷歌云GCE后，点击”free trial”
  2. 点击“转至控制台”
  3. 左上角菜单选择“compute engine”，选择“VM 实例”，点击“创造实例”。完成后，“外部IP”列会出现IP地址。
    • 输入network tag configed in firewall rule: like tagin, tagout
    • allow http/https access
    • [optional] pubkey for SSH connect from putty/xshell etc (参考)
