Website To Image

Apr 6, 2019


背景：

当本地静态网页(如Jekyll)包含众多元素，而不仅仅是文字时，最简单直接、最精简的方式是转换成“图片”，尤其便于分享。

插件方案 (不推荐)：

chrome浏览器，插件：“小白工具包”，但不知何故，转换后的图片或pdf 均会将原网页表格的board丢失掉，因此不推荐该方案。


浏览器自带转换工具：

Maxthon遨游浏览器，自带工具：“菜单” → “另存为” 图片或其他，完美转换。
