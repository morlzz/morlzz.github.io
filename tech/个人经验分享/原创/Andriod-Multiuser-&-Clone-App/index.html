Android MultiUser & Clone App

Jun 2, 2019

Category▸  Tech   个人经验分享   原创
Tag▸  Android   MultiUser&CloneApp

Abstract ▹ 安卓多用户、克隆App


Note: 以下经验为亲测原创，一方面作为日常总结纪录，另分享给有兴趣的人。


作用

安卓多用户、克隆App，在日常使用中需求愈来愈明显。

安卓多用户：等于sandbox，有效隔离各个用户的空间。

  • 安卓自带多用户，需切换至不同用户
  • 另一种方式 AFW (Android For Work), 无需切换用户，该用户与主用户界面共存；
    界面呈现上，”Work(工作资料)”沙箱空间与主用户同时工作，逻辑上两者还是多用户概念。
    • 推荐谷歌商店App: Island

克隆App：适用于多个app账号（亲测，有些app如手机银行登录时不识别克隆的app）。

  • 推荐谷歌商店App: App Cloner


安卓多用户

操作时几个要点：

  1. 主用户、其他用户：各自安装维护删除第三方App，互不影响。
    • 系统App：慎删、慎冻结，会影响正常使用这些App的用户。
  2. 主用户更新App时：
    • IF 其他用户也安装了相同App：其他用户自动更新对应App。(反之，理论上：其他用户更新，主用户也应该自动更新，未测。)

  3. 使用钛备份Titanium Backup恢复App及其数据时：所有其他用户都会自动安装对应App (其他用户仅是恢复App，数据无法恢复)。

  4. 权限/空间管理软件推荐：App Ops、MyAndroidTools、Storage Redirect 存储重定向。
    • App Ops、Storage Redirect 存储重定向: 支持在当前用户下跨多用户管理
    • MyAndroidTools: 仅支持当前用户管理
    • 此外，钛备份Titanium Backup：仅支持当前用户管理 (但恢复时，会恢复安装至所有用户下)
  5. 备份软件：钛备份Titanium Backup、一加搬家 OneplusBackuprestore (一加手机自带专用)
    • 钛备份Titanium Backup：支持app及其数据的备份恢复。
    • 一加搬家 OneplusBackuprestore：
    • 仅支持app软件本身的备份恢复，不支持对应的应用数据的备份恢复。
    • 优点：可备份恢复基本系统配置：如联系人、桌面图标设置、壁纸、铃声、wifi等。


App克隆

  1. App Cloner的操作仅对当前用户生效，不会安装到其他用户下。

  2. 主程序、克隆程序：

    • 软件版本互不影响
    • 主程序更新后，App Cloner会提醒”克隆程序有更新
