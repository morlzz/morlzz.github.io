For Linux (Test in Ubuntu 18.04)

安装最新的Android-platform-tools

Step1: 安装adb

$ sudo apt-get install adb                //卸载 sudo apt purge -y adb && sudo apt autoremove -y

$ adb version                             //查看安装信息
Android Debug Bridge version 1.0.39
Version 1:8.1.0+r23-5~18.04
Installed as /usr/lib/android-sdk/platform-tools/adb

Step2: 将platform-tools-latest-linux.zip解压缩后，直接拷贝至/usr/lib/android-sdk/platform-tools/目录

$ adb kill-server                          //否则当adb server正在运行时，会无法覆盖adb文件
$ sudo cp -r ./ /usr/lib/android-sdk/platform-tools

Step3: 再次检查adb version

$ adb version                             
Android Debug Bridge version 1.0.41
Version 29.0.6-6198805
Installed as /usr/lib/android-sdk/platform-tools/adb

 

为adb加入path

参考

$ ls -la ~/ | more                          //".bashrc"为隐藏文件，输入该命令可以看到

$ sudo nano ~/.bashrc                       // 编辑或创建(如原先没有)该文件

//在弹出的".bashrc"文件中，加入下行：
export PATH=${PATH}:/usr/lib/android-sdk/platform-tools/adb

$ adb                                       //检查是否工作；如果不工作，安装"glibc.i686"和"libstdc++"

 

其他命令

$ adb kill-server
$ adb start-server
