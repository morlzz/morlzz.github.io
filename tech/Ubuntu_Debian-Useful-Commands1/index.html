『Debian/Ubuntu终端命令』Debain Terminal Commands

 

#安装.deb包

$sudo apt install ./name.deb

#显示安装包

$ dpkg --list                        #show packages list

#卸载安装包xxx的几种方法

$ sudo apt-get --purge remove xxx    #方法1：remove package "xxx", "–-remove"指卸载安装, "–-purge"指清除配置
$ dpkg --remove xxx                  #方法2
$ dpkg -r xxx.deb                    #方法3
$ sudo apt purge -y packagename && sudo apt autoremove -y    #方法4

  • 例如，卸载安装包yyy，全过程

      #卸载yyy
  sudo apt remove --auto-remove yyy                  #删除已安装的软件包
  sudo rm -f /etc/apt/sources.list.d/yyy.list        #禁用存储库

  #进一步清理		
  # apt-get clean && apt-get autoremove
  $ sudo apt-get -f install              #Another command to fix dependency problems
  $ sudo dpkg --configure -a             #checks for dependency problems to fix            
  $ sudo apt-get clean                   #clean unnecessary dependencies, "apt-get autoclean"
  $ sudo apt-get update                  #Updates the packages’ list in the repositories

#Removing Broken Packages

  • clean: Removes cache of programs older than the installed.
  • autoremove: Removes unnecessary files, like dependencies which are not longer needed.
  • -f / –fix-broken install: Fix broken dependencies, correct possible package corruption problems.
# apt-get clean && apt-get autoremove
$ sudo apt-get -f install
$ sudo dpkg --configure -a

#Troubleshoot for issues when installing or uninstalling software

$ sudo apt-get update                  #Updates the packages’ list in the repositories
$ sudo dpkg --configure -a             #checks for dependency problems to fix
$ sudo apt-get -f install              #Another command to fix dependency problems
$ sudo apt-get clean                   #clean unnecessary dependencies, "apt-get autoclean"

#Check if packages are being held by the packages manager

# apt-get -u dist-upgrade                        #To check if the installer holds packages pending of installation run
# apt-get remove –dry-run packagename            #to remove the be held packages, to run

#tar.gz文件解压缩

tar xvzf 1.tar.gz

#tar.bz2文件解压缩

tar xvjf 2.tar.bz2

#zip文件解压缩

sudo apt install unzip
unzip test.zip                 #把文件解压到当前目录下
unzip -d /temp test.zip        #把文件解压到指定的目录下
