Windows 10 Sandbox ".wsb" File Config

Apr 7, 2020

Category▸  Tech   搬运
Tag▸  OS



设置教程
微软官方.wsb设置说明

 

  • 例如：RunSandbox_MyConfig.wsb (在host system中，双击wsb文件，即可直接打开设置后的sandbox)

      <Configuration>
    <MappedFolders>
      <MappedFolder>
        <HostFolder>C:\Users\yourusername</HostFolder>
        <ReadOnly>true</ReadOnly>
      </MappedFolder>
    </MappedFolders>
	  
    <LogonCommand>
        <Command>explorer.exe C:\Users\WDAGUtilityAccount\Downloads</Command>
    </LogonCommand>  
  </Configuration>

  • Sandbox 默认用户名： WDAGUtilityAccount

    • 路径：Sandbox中 C:\Users\WDAGUtilityAccount\
