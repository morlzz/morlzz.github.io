Jekyll Post be hidden But can be accessible

参考以上链接方法，在Post YAML frontmatter中设置”visible变量” (名字随意定义)，思路：

  • 当visible= 0时，该post不会显示在首页列表中，但可被其他post引用。(对Category/Tag功能、Search功能完全不影响。)
    • 全局自带的published: true或是published: false, false启用后, post完全不可被调用读取。
  • 配合”visible变量”方法，还需在\_layouts\home.html中的forloop中加入代码:

    {%- for post in site.posts -%}
	  {%- if post.visible != 0 -%}	
      ... 
	  {%- endif -%}
    {%- endfor -%}


.gitignore使用方法

原出处

注释方法 被忽略的类型
*.a * 代表所有, 即忽略所有 .a类型结尾的文件
/todo 仅仅忽略 RootSRC(项目根目录)/todo, 即忽略指定的文件或文件夹(不包括子目录)
todo 忽略 todo/ 目录下的所有文件, 包括子目录
doc/*.txt 会忽略 doc/ 目录内的所有.txt文件, 但不包括子目录的
