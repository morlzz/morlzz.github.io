启用Ubuntu Touch的ssh功能

Feb 20, 2019

Category▸  Tech   搬运
Tag▸  OS

Abstract ▹ 远端和主机的连接

Ubuntu OS的移动版本 -Ubuntu Touch，由 UBports 社区制作和维护。

Ubuntu Touch官方支持的移动设备：https://devices.ubuntu-touch.io/

官方App：https://open-store.io/


SSH for Ubuntu Touch

启用Ubuntu Touch的ssh功能，官方方法：

http://docs.ubports.com/en/latest/userguide/advanceduse/ssh.html


1. [Create your public key from remote, eg. laptop]

ssh-keygen

（id_rsa为私钥，id_rsa.pub为公钥；导入主机（Ubuntu Touch手机）的为公钥；使用ssh软件（如xshell），需采用“公钥验证”方式&导入私钥以生成公钥，用户名“phablet”）

2. [Copy the public key to phone]


3. [Configure phone] (assume public key stored as /home/phablet/id_rsa.pub)

3.1 [terminal app on phone]

mkdir /home/phablet/.ssh

chmod 700 /home/phablet/.ssh

cat /home/phablet/id_rsa.pub >> /home/phablet/.ssh/authorized_keys

chmod 600 /home/phablet/.ssh/authorized_keys

chown -R phablet.phablet /home/phablet/.ssh


3.2 [Now start the ssh server on phone]

sudo android-gadget-service enable ssh


4. [Connect]

ssh phablet@ < ip-address >
