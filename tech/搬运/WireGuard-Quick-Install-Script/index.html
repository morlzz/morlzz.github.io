WireGuard 一键安装脚本

Jan 7, 2019

Category▸  Tech   搬运
Tag▸  Cloud/VPN/VPS

Script for Debian 9/Ubuntu/Centos_7

原出处
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
# 一键安装wireguard 脚本 Debian 9 (源:逗比网安装笔记)
wget -qO- git.io/fptwc | bash

# 一键安装wireguard 脚本 Ubuntu   (源:逗比网安装笔记)
wget -qO- git.io/fpcnL | bash

# CentOS7一键脚本安装WireGuard   (https://atrandys.com/2018/886.html)
yum install -y wget && wget https://raw.githubusercontent.com/atrandys/wireguard/master/wireguard_install.sh && chmod +x wireguard_install.sh && ./wireguard_install.sh
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
重启Linux系统：

重启vps系统： reboot （直接输入，回车）

CentOS重启ssh：/etc/init.d/sshd restart

Debian/Ubuntu重启ssh：/etc/init.d/ssh restart

重启Nginx： /etc/init.d/nginx restart

重启PHP-FPM：/etc/init.d/php-fpm restart

重启pptp： /etc/init.d/pptpd restart

重启Mysql：/etc/init.d/mysql restart（有时候需要手动加载mysql进程：/etc/init.d/mysql reload）
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Special for Ubuntu_AWS, Steps:

1. optional: 创建EC实例，“安全组”增加TCP和UDP，指定一样的“端口”，允许 “0.0.0.0/0”。

2. # 一键安装wireguard 脚本 Ubuntu   (源:逗比网安装笔记)
wget -qO- git.io/fpcnL | bash

3. 使用“ bash wgmtu”  命令，按“3”，服务器设置与上面一样的“端口”；再按“5”，自动更新客户端配置，其中"client.conf“不会更改，需手动更改为与上面一样的”端口“。

4. 启动服务端
wg-quick up wg0 
   （关闭为：wg-quick down wg0）
可以通过命令 wg 检查启动是否成功，成功的话会输出如下内容： 
interface: wg0
  public key: xxxxxxxxxx
  private key: (hidden)
  listening port: 10443
5. 将 WireGuard 设置成开机启动
$ systemctl enable wg-quick@wg0
6. 额外的配置
为了让VPN能正常工作，需要启用ipv4的ip_forward功能
$ sysctl -w net.ipv4.ip_forward=1
7. 重启ssh：
/etc/init.d/ssh restart
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Notes for Debian_GCE:

1. 一键脚本。
   【Optional】wg配置： 可设置指定端口（同上, "client.conf“需手动更改）
    Plus: 以上步骤 4~7.
2. firewall rule: network tags: http-server, httpw-server, in, out
    (优先级 1，IP Range: 0.0.0.0/0，port/protocol: all)
    http-server, httpw-server: 创建vm实例时，tick "allow http" & "allow https". 
    in, out: 自行创建，in for "ingress",out for "outgress".
