TunSafe install on Linux

Dec 30, 2018

Category▸  Tech   搬运
Tag▸  Cloud/VPN/VPS

Official Guide

Install

$ git clone https://github.com/TunSafe/TunSafe.git
$ cd TunSafe

$ sudo apt-get install clang-6.0
$ make
$ sudo make install


Run

$ sudo tunsafe start -d ./home/zhao/TunSafe/wg_mm.conf
$ sudo tunsafe show
$ sudo tunsafe stop tun0
