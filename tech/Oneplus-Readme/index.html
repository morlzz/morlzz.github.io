一加手机使用相关说明

Aug 13, 2019

Category▸  Tech
Tag▸  OS


1. “系统App”进行”固件更新”后的软件版本

假设系统固件更新前，”系统App”的版本是”A”，新固件中内置的”系统App”的版本是”B”，”A”高于”B”，那么，完成固件更新后，”系统App”的版本仍然是”A”。

 

2. 官方备份恢复工具：”一加搬家 OnePlus Swich”

一般主要使用其：”联系人”、”一加桌面”、”系统设置” 备份功能。而Apps的备份及其数据迁移，”一加搬家”并不支持，推荐使用”钛备份”。
