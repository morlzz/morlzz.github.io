Install WSL2 & Backup/restore WSL2

Apr 4, 2020

Category▸  Tech
Tag▸  WSL



背景：WSL2支持原生Linux Kernel, 即可实现安装浏览器等。

 

微软官方教程: EN, CN

 

Step1：加入 Windows 预览体验计划

WSL 2 仅适用于 Windows 10 版本 18917 或更高版本。因此，需加入 Windows 预览体验计划, 选择“快速”或“慢速”升级。

  • CMD窗口：ver命令, 检查 Windows 版本。

 

Step2：启用“虚拟机平台”可选组件并确保启用了 WSL

  • 在 PowerShell 中运行以下命令: 
      dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
  dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart
  • 重启计算机来完成以上两个组件的安装。

 

Step3：确保已从Microsoft Store安装WSL1

  • My Case: WSL1 - Ubuntu 18.04 LTS
  • 安装完成后，需进入初始设置UNIX用户名和UNIX密码。

 

Step4：更新为WSL2 Linux内核

  • 安装文件_PC-x64

 

Step5：使用PowerShell设置要由 WSL2 支持的发行版

  • 使用wsl -l，显示当前发行版的名称，如Ubuntu 18.04。
  • 设置发行版，运行：wsl --set-version <Distro> 2，其中更改为`Ubuntu 18.04`
    • wsl --set-version Ubuntu-18.04 2
  • 如要使 WSL 2 成为默认体系结构，运行：wsl --set-default-version 2，这会使安装的任何新发行版均初始化为WSL2发行版。
  • wsl --list --verbose或者wsl -l -v，显示有关发行版所有的详细信息：每个发行版名称、状态、版本号、正在运行的版本。默认发行版标注星号。

 

Backup/restore WSL2

使用Powershel自带”wsl –export”、”wsl –import”命令直接进行备份和恢复生成的”.tar”文件。

//To list the distros installed
wsl --list

//to export a distro’s file system to a TAR file:
wsl --export distro_name file_name.tar
wsl --export Ubuntu-18.04 MyDebianBackup20200416.tar
wsl --export Debian MyDebianBackup20200416.tar


//to import the TAR backup file
wsl --import distro_name install_location file_name.tar
wsl --import Ubuntu-18.04 C:\Downloads\myinstall C:\1\MyDebianBackup20200416.tar
wsl --import Debian C:\Downloads\myinstall C:\1\MyDebianBackup20200416.tar

注销WSL2

和卸载效果一样。

wsl --unregister Debian
wsl --unregister Ubuntu-18.04

改变登录WSL2的默认用户名

假设 WSL2-ubuntu 18.04。

ubuntu1804 config --default-user <username>

 

WSL2 相关路径

windows host可直接浏览wsl完整文件的路径

  • 在文件浏览器中输入： 
      \\wsl$
  • WSL2-Ubuntu 默认安装路径： 
      C:\Users\<username>\AppData\Local\Packages\CanonicalGroupLimited.Ubuntu18.04onWindows_...\LocalState
  • WSL2-Debian 默认安装路径： 
      C:\Users\<username>\AppData\Local\Packages\TheDebianProject.DebianGNULinux_...\LocalState
