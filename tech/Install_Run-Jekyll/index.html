Install & Run Jekyll

Apr 11, 2020

Category▸  Tech
Tag▸  Jekyll

 

安装教程

Step1: 安装Ruby

  • RubyInstallers下载：Ruby+Devkit(x64).exe
    • 安装；勾选”添加到PATH”选
    • 勾选”安装msys2”
    • 选择 “3 – MSYS3 and MINGW development toolchain”

 

Step2: 安装RubyGems

  • RubyGems下载：RubyGems
    • For Windows, download “.zip” file.
    • 在CMD或者Powershell命令行： 
        cd {unzip-path} // unzip-path表示解压的路径
  ruby setup.rb

 

Step3: 安装Jekyll

//在CMD或者Powershell命令行
gem install jekyll

gem install bundler
bundle install                               //to install missing gems

 

Step4: 安装Jekyll-paginate

//在CMD或者Powershell命令行
gem install jekyll-paginate

 

Step5: 验证安装是否完成

//在CMD或者Powershell命令行
jekyll -v
gem -v
ruby -v

 

Step6: 用jekyll命创建博客模板

//在CMD或者Powershell命令行
cd xxx  
jekyll new mytestblog
cd mytestblog

 

Step7: 本地运行博客

  • 方法1 (不推荐)

      //在CMD或者Powershell命令行
  jekyll serve                             //runs some version of jekyll server, which may or may not be the one you want  
  jekyll serve --drafts

  • 方法2 (推荐)

      //在CMD或者Powershell命令行
  bundle exec jekyll serve                 //runs the exact jekyll server version that is specified in your Gemfile/Gemfile.lock 
  bundle exec jekyll serve --drafts        //含"drafts"目录

 

Step8: 浏览器浏览

  • 查看_config.yml文件中的端口”Serving Host/Port”，默认127.0.0.1:4000。
  • 浏览器输入: http://127.0.0.1:4000/ 或者 http://1ocalhost:4000/,即可。

 

Issues

  • jekyll serve启动报错

    往往是因为端口被其他应用占有导致，一般需要kill relative port for the other app, 或者重启计算机。有时，更改了”_config.yml”中的端口，还是启动报错，往往重启才能解决问题。

    • 最快的方式 - 指定端口, CMD 命令行： 
        bundle exec jekyll serve -P <Your_Assigned_New_Port>
