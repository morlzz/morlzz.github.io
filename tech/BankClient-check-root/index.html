银行App绕过Root检测 (如建设银行App)

背景：对于已经Root的手机，部分App严苛检测，一经检测到，完全不让使用该App。

Root (Magisk)手机: 绕过Root检测的方法 (20190619 亲测)

  1. Magisk Manager: 安装 MagiskHide Props Config、Riru-Core、Riru-EdXposed模块。

  2. EdXposed Manager: 安装 对话框取消、HiddenCore Module、RootCloak、XPrivacyLua模块。
    • XPrivacyLua：禁止 “读取应用列表”。
    • RootCloak：将App”包名”加入。
    • 对话框取消：开启增强模式，在目标App设置中启用”防检测”，”隐藏其他应用”中勾选所有使用到Root权限的App。
    • HiddenCore Module：无需任何设置。

    Note：if < Android 9，use “Xposed framework/manager” directly, no need install “EdXposed” module through Magisk.

  3. 在 Magisk Manager “Magisk Hide”中，勾选对应App。

  4. 重启即可。


Note:
1) 理论上所有的类似App (尤其: 银行类App) 均可适用该方法。
2) 类似App需要安装在其他用户/AFW下，步骤如下：
 - 在主用户上，先行安装该App、并使用以上Magisk + EdXposed方法；
  * 该App：不用打开，禁止任何权限。
 - 在主用户上，确保设置好Magisk Hide后，可将该App直接”禁用”或者”冻结”；
 - 尔后，在其他用户或者AFW上，安装该App，则可，绕开效果和主用户上一致。
  * 该App：在其他用户或者AFW下，给予所需权限，以便正常使用。
